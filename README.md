# pw-course

Dự án học Playwright automation testing.

## Mô tả

Dự án này chứa các test cases sử dụng Playwright để test trang web https://material.playwrightvn.com/

## Cài đặt

```bash
npm install
```

## Chạy tests

```bash
# Chạy tất cả tests
npm test

# Chạy tests với UI mode
npm run test:ui
```

## Tests

- `has title`: Kiểm tra title của trang web
- `get started link`: Kiểm tra navigation và heading sau khi click link
